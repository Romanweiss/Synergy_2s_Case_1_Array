# Кейс-задача №1 — Работа с одномерным массивом (Python)

## Описание
В рамках кейс-задачи реализована программа на языке Python для работы с одномерным массивом целых чисел.  
Программа формирует массив случайной размерности и вычисляет сумму отрицательных элементов, расположенных между минимальным и максимальным значениями массива.

## Условие задачи
Дан одномерный массив `A` размерности `N`.  
Необходимо найти сумму отрицательных элементов, расположенных между минимальным и максимальным элементами массива.

## Реализация
- Массив реализован с использованием стандартного модуля `array`
- Размерность массива `N` генерируется случайным образом в диапазоне от **10 до 30**
- Значения элементов массива генерируются случайным образом в диапазоне от **−1000 до 1000** (включительно)
- Для определения диапазона используется поиск индексов минимального и максимального элементов

## Используемые технологии
- Python 3.x
- стандартный модуль `array`
- стандартный модуль `random`

## Алгоритм работы
1. Случайным образом определяется размерность массива
2. Формируется одномерный массив целых чисел
3. Находятся минимальный и максимальный элементы массива и их индексы
4. Определяется диапазон элементов между ними
5. Вычисляется сумма отрицательных элементов в данном диапазоне
6. Результаты выводятся в консоль

## Примечание

Каждый запуск программы формирует новый набор данных, что позволяет использовать решение для проверки корректности алгоритма при различных входных значениях.

## Запуск программы
Для запуска программы выполните команду в терминале Powershell, либо Bash, находясь в папке с файлом скрипта main.py:

python main.py
````
