# Кейс-задача №1 — Работа с одномерным массивом (Python)

## Описание
Программа формирует одномерный массив случайной размерности и вычисляет сумму отрицательных элементов, расположенных между минимальным и максимальным элементами массива.

Теперь сценарий работы интерактивный:
1. Пользователь сначала выбирает тип чисел в массиве.
2. После выбора запускаются генерация данных и расчёты.

Поддерживаемые режимы:
- `int` — массив целых чисел.
- `float` — массив вещественных чисел.
- `mixed` — смешанный массив, где в одном массиве могут быть и `int`, и `float`.

## Важный момент по типам данных
- `array` хранит только один тип данных на весь массив (`typecode`).
- Поэтому для `int` и `float` используется `array`.
- Для `mixed` используется `list`, так как список Python поддерживает смешанные типы.

## Условие задачи
Дан одномерный массив `A` размерности `N`.
Необходимо найти сумму отрицательных элементов, расположенных между минимальным и максимальным элементами массива.

## Реализация
- Размерность массива `N` генерируется случайно в диапазоне от **10 до 30**.
- Значения генерируются в диапазоне от **-1000 до 1000**.
- Для `float` числа округляются до 2 знаков после запятой.
- Границы суммирования определяются индексами первого вхождения минимального и максимального элементов.
- Суммируются только отрицательные элементы между этими индексами.

## Используемые технологии
- Python 3.x
- Стандартные модули: `array`, `random`

## Структура проекта
- `main.py` — основной скрипт с логикой выбора типа, генерации данных и расчётов.
- `README.md` — инструкция по запуску и описание решения.
- `task_solution_description.txt` — подробное описание процесса решения и структуры программы (для отчёта).

## Запуск программы
```bash
python main.py
```

## Как работает программа
1. На экране появляется меню выбора типа чисел.
2. Пользователь вводит `1`, `2` или `3`.
3. Программа проверяет корректность ввода.
4. После успешного выбора генерирует массив.
5. Находит минимум, максимум и их индексы.
6. Считает сумму отрицательных элементов между ними.
7. Выводит результат в консоль.
